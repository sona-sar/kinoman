# KINOMAN - the trivia game for movie lovers 🍿

Kinoman is a trivia game written in React JS, where players are tasked to guess an actor from the movie that is presented by its poster. If you guess correct you gain +100 points, if you guess wrong you lose the entire game. You can use hints (-50 points) or skip a question (-100 points). If you guess 2 questions correct in a row you get to play another trivia game where you choose the category. If you guess correct you get +200 points and if you guess wrong you don't lose any points. 

## Getting Started

Follow these steps to set up and run the Kinoman project on your local machine.

### Prerequisites

- Node.js and npm (or yarn)
- Visual Studio Code (or any preferred code editor)
- No need for API keys since those are already there (not a good practice)

### Installation

1. Clone the repository:
   `git clone https://github.com/sona-sar/kinoman.git`
2. Open the project in Visual Studio Code
3. Install dependencies:
   `yarn`
4. Run the project:
   `yarn start`
